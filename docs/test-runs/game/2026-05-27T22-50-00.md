- **Session expired path observed twice**: (1) Tab 3 opened with Beta's stored credentials while Beta was still live — server rejected reconnect because player wasn't disconnected, "Your session expired" shown correctly; (2) post-browser-close reload — all tabs died simultaneously, new tab tried to reconnect with Beta's credentials, race condition caused server to reject, "Your session expired" shown again.

- **localStorage collision (test artifact only)** — all browser tabs share the same `localStorage` origin, so Tab 3's join attempts (Gamma, Latebird) overwrote the real Beta player UUID with a temporary UUID. This required an explicit `localStorage.setItem(myId, gameCode)` on Beta's tab before each test requiring reconnect. On real mobile use, each player has their own device/localStorage; this collision cannot occur in production.

- **`browser_close` closes all tabs** — closing Tab 2 via the Playwright MCP `browser_close` tool closed the entire browser session, not just that tab. This is a test tooling limitation. Future disconnect tests should use `ws.close()` via `evaluate` to simulate disconnect without killing the tab.
