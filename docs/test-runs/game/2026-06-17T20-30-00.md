## Findings

- **`WEBAUTHN_RP_ID` in dev `docker-compose.yml` is `tensies.app`**, not `localhost`. This blocks WebAuthn ceremonies in local Playwright tests (the browser rejects the RP ID as not matching the current domain). The 2026-06-15 run worked because it manually overrode `WEBAUTHN_RP_ID=localhost` when restarting the stack. Steps 25/29 fell back to JWT injection this run.
- **End Game feature works end-to-end.** New in this branch: tap-to-confirm UX (first tap shows "Tap to confirm", second sends `end_game`), overlay with scoreboard on both clients, "Bummer!" dismiss returns to landing. SessionStorage persistence not tested (single-page session).
