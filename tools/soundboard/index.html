TENSIES

Audio Share Testing Soundboard

Tensies has an experimental audio share feature — the host's phone chirps the game code as a short melody and a joining player's phone decodes it through the mic. It works, but it sounds like a modem. This soundboard helps iterate on making it musical, catchy, and pleasant. Pick a voice, hit Play to hear it, or Test to run a real speaker-to-mic decode and see if it still lands.

Encoder/decoder constants are mirrored from static/js/audio-share.js — if the production FSK scheme changes, update this file to match. See README.md for the full list.
